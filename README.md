# Age Calculator

A simple, fast age calculator tool that calculates your exact age in years, months, days, hours, and minutes. Also shows interesting facts about your life and upcoming birthday countdown.

## Features

- Calculate exact age from birthdate
- Support multiple date input formats
- Show age in different units (years, months, days, hours, minutes)
- Birthday countdown
- Interesting life statistics (heartbeats, breaths taken, etc.)
- Zodiac sign information
- Mobile-responsive design
- No tracking, no ads, completely free

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- date-fns for date calculations

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel with automatic deployments from the main branch.
